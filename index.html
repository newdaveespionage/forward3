Raspberry Pi

  • Inexpexnsive, self-contained, hackable computer platform (runs varieties of linux)
  • Can handle hardware hooked up directly to it's board via programmable pins
  • Many accessories for extending it's functionality

PubNub

  • SaaS provider of lightweight messaging managment (32k max message size).
  • Built-in analytics, debugging tools, API's
  • Useful for connecting many independent self-aware robots
